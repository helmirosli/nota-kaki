# NOTA KAKI — Project Plan

> Interactive animated study notes for kids aged 7–12, powered by JAIS Islamic education PDFs.

---

## 1. Vision

Transform dry religious textbook PDFs into a **magical, animated learning experience** for kids. Each note feels like a storybook — colourful cards, bouncy animations, friendly mascots, and mini-games. Kids navigate like an explorer discovering treasure maps, not reading a textbook.

## 3. User Journey (Navigation Flow)

```
/dashboard
 └── [JAIS] card
      ├── [Tahun 1] card
      │    ├── [Akhlak] subject card
      │    │    ├── Chapter list (animated cards)
      │    │    │    └── /note/[id]  ← animated note page
      │    ├── [Bahasa Arab]
      │    ├── [Feqah]
      │    ├── [Jawi]
      │    ├── [Hafazan]
      │    └── [Tauhid]
      └── [Tahun 2] card
           └── (same subjects)
```

Each level has its own entrance animation:
- Dashboard → floating island cards flying in
- Tahun level → colourful door-open transition
- Subject → cards fan out like a deck
- Note → page-flip / scroll reveal per section

## 6. Visual Design Language

- **Palette**: Warm greens, golden yellows, soft purples — Islamic-inspired but child-bright
- **Typography**: Rounded sans-serif (Nunito or Baloo 2) — friendly, legible
- **Cards**: Large rounded corners (radius 2rem), soft drop shadows, subtle gradients
- **Mascot**: A friendly animated crescent moon + star character that reacts to progress
- **Backgrounds**: Animated particle fields (floating dots, stars) using Framer Motion
- **Icons**: Bold emoji + custom SVG Islamic geometric patterns as decorative accents

---

## 7. Key Features

### A. Animated Dashboard
- Floating 3D-style cards for each source (JAIS)
- Hover: card lifts and glows
- Click: card "opens" with scale + blur transition revealing year cards

### B. Subject Cards
- Each subject has a unique colour and emoji icon
- Progress ring showing chapters completed
- "New" badge with pulse animation for unread chapters

### C. Animated Note Pages
- Sections reveal on scroll (Framer Motion `whileInView`)
- Concept cards flip in from sides
- Illustrations: Lottie animations embedded per section
- "Read to Me" button (Howler.js + Web Speech API)
- Swipe-left/right between chapters

### D. Knowledge Graph
- At the bottom of each note, a mini interactive graph (React Flow)
- Nodes = key concepts, edges = relationships
- Tap a node to highlight its definition

### E. Mini Quiz
- 3–5 questions after each chapter
- Immediate animated feedback (bounce ✅ or shake ❌)
- Stars awarded (1–3) on completion
- Confetti burst on 3 stars

### F. Progress Dashboard
- Star collection per subject
- Animated progress bar
- "Trophy room" for completed subjects

## 8. PDF → Note Pipeline (Script)

```
scripts/
  extract-pdf.ts     → reads PDF, extracts text per page
  generate-notes.ts  → sends text to Claude API, receives structured JSON
  save-notes.ts      → writes JSON to /src/data/notes/[id].json
  run-pipeline.ts    → orchestrates all 3 steps for all PDFs
```

Run once to generate all notes:
```bash
npx tsx scripts/run-pipeline.ts
```
